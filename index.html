Features

Everything a modern developer needs.

No Electron. No subscriptions. Just a fast, focused native app.

AI Agents, natively

Run Claude Code, opencode, Codex, Gemini CLI, or Aider inside a real PTY — auto-resolved to the right directory, no PATH wrangling.

Worktree-first

Each worktree is a full session: agent, shell, file tree, and diff all follow the active checkout. Switch branches without losing context.

Git panel built in

Branch switch, fetch, pull, push, and stage-all commit. A recent commit log shows where you left off. No terminal needed.

File tree + Diff

Lazily-expanding file browser with icon grid view. Working-tree diff against HEAD with per-file stats, rendered unified or side-by-side.

Shell tab

An interactive login shell in the exact same directory as your agent — run a test, check a log, copy a path.

No destructive defaults

Force push, branch delete, and discard are deliberately absent. Every outward-facing action confirms before it runs.

Keeps working while you don't

Switch to another workspace and its agent and shell keep running in the background — Grove pools the underlying processes independently of whatever's on screen. Check on one project without pausing another. Come back, and it's on the tab you left it on.
